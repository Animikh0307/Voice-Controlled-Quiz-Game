# Voice-Controlled Quiz Game

An interactive quiz game built using Python that allows users to answer questions through voice input. The game combines a graphical interface with multimedia elements such as background video, sound effects, and real-time feedback to create an engaging and immersive experience.

## Features

* Voice-based answer input for hands-free interaction
* Dynamic question flow with randomized questions
* Real-time score tracking and feedback
* Background video rendering for enhanced visuals
* Sound effects for user actions and game events
* Simple and intuitive user interface

## Overview

The game presents a series of questions to the user, who can respond using voice commands. The system processes the input, evaluates the response, and updates the score accordingly. Visual and audio feedback is provided throughout the gameplay to improve user engagement and interaction.

## Tech Stack

* Python
* Tkinter (GUI)
* Pygame (audio handling)
* ImageIO (video processing)
* SpeechRecognition (voice input)
* pyttsx3 (text-to-speech)

## Use Case

This project demonstrates the integration of voice recognition with interactive applications and can be used as a foundation for building more advanced educational tools or voice-driven interfaces.
